## What is Mubboo?

Mubboo is a multi-country AI life assistant that helps everyday consumers make better decisions across five key areas of life:

- 🛍️ **Shopping** — Product comparisons, price tracking, and buying guides
- ✈️ **Travel** — Flight search, destination guides, and travel planning
- 💰 **Finance** — Banking comparisons, insurance guides, and financial literacy
- 📍 **Local** — Restaurant guides, local services, and city recommendations
- 📖 **Info** — Government services, how-to guides, and everyday knowledge

Every country site is built with native-quality local content — real brands, local prices, actual regulations, and cultural context. No machine translation. No generic global content.

## This Repository

This is the source code for [mubboo.ai](https://mubboo.ai), the global brand hub for the Mubboo platform. It serves as:

- **Brand presence** for investors, media, and partners
- **News channel** with daily coverage of AI, e-commerce, travel, finance, and consumer technology
- **Blog** with monthly deep dives on industry trends and platform updates
- **Link hub** connecting all country sites

## Tech Stack

- **Framework:** Next.js (App Router)
- **Deployment:** AWS Amplify
- **DNS:** Cloudflare
- **Content:** MDX articles with dynamic sitemap generation
